This repo contains R markdown files for:
* scRNA-seq Analysis (Seurat) of in vivo (mouse) generated hybrid fusion cells
* scATAC-seq Analysis (ArchR) of in vivo (mouse) generated hybrid fusion cells
* scRNA-seq Analysis (Seurat) of artificial doublet experiment
* scRNA-seq Analysis (Seurat) of patient-isolated hybrid fusion cells (SmartSeq)
* scRNA-seq Analysis (Seurat) of patient-isolated hybrid fusion cells (10xGenomics)

Additional analysis included for copy number estimation, tumor module generation, etc.

Furthermore, how Cell Ranger was utilized alongside a homebrew workflow for pre-processing 10x and SmartSeq data respectively is are also documented. Processed files were readied to be analyzed in Seurat.
